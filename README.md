# Практична робота "Реалізація успадкування"

Цей репозиторій містить виконане завдання з наслідування в Java. В якості моєї тварини я обрав **лінивця**.

## Обрана тварина
- Тварина: `Sloth` (лінивець)
- Успадкування: `Sloth` наслідує загальні властивості від `Animal`

## Результат роботи
- `src/domain/Animal.java` — базовий клас для тварин
- `src/domain/Sloth.java` — клас лінивця з унікальними методами та властивостями
- `src/test/TestAnimal.java` — тестовий клас, який створює лінивця і викликає його методи

## Поведінка лінивця
- `climb()` — повільно піднімається на дерево
- `hang()` — висить на гілці і відпочиває
- `sleep()` — спить багато годин
- `eat()` — їсть листя повільно
- `speak()` — видає тихі звуки

## Як запускати
1. Скомпілюйте класи:
   ```bash
   javac src/domain/*.java src/test/TestAnimal.java
   ```
2. Запустіть тестовий клас:
   ```bash
   java -cp src test.TestAnimal
   ```

## Примітка
Ця робота показує, як можна використати успадкування для моделювання загальних властивостей тварин і додати конкретну поведінку для обраного виду.
